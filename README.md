# CamScanner Plugins

CamScanner Claude Code 插件市场，提供 YApi、i18n 等插件。

## 安装

```bash
# 1. 添加市场（仅首次）
claude plugin marketplace add tianmuji/camscanner-plugins

# 2. 安装插件
claude plugin install yapi@camscanner-plugins
claude plugin install i18n@camscanner-plugins
```

## 包含插件

| 插件 | 说明 |
|------|------|
| **yapi** | YApi API 文档管理（查询、创建、更新、导出接口文档） |
| **i18n** | 多语言字符串集成（搜索、导出、写入多语言） |
